# VAREL — everyday style

> 🎉 **Проект готовий до запуску!** Всі залежності встановлені, база даних налаштована, дані імпортовані.

## 🚀 Швидкий запуск

**Просто двічі клацніть на файл:**
```
START_SERVER.bat
```

**Або вручну:**
```bash
cd backend
.\.venv\Scripts\python.exe manage.py runserver 127.0.0.1:8000
```

Потім відкрийте у браузері: **http://127.0.0.1:8000**

## 🔑 Доступ до адмін панелі

- `START_SERVER.bat` — Запуск сервера (Windows)
- `ІНСТРУКЦІЯ_ЗАПУСКУ.md` — Детальна документація

## 🌐 Доступні сторінки

- `/` — Головна сторінка
- `/catalog` — Каталог товарів
- `/product/<id>` — Сторінка товару
- `/about` — Про нас
- `/contacts` — Контакти
- `/admin` — Адмін панель

## 📡 API Endpoints

| Метод | URL | Опис |
|-------|-----|------|
| GET | `/api/products/` | Всі товари |
| GET | `/api/products/?category=Джинси` | Фільтр за категорією |
| GET | `/api/products/<id>/` | Один товар |
| POST | `/api/orders/` | Створити замовлення |
| GET | `/api/orders/list/` | Список замовлень |

## 🛠️ Архітектура

Django-моноліт з вбудованим REST API.  
**Frontend**: Django-шаблони + Vanilla JavaScript.  
**Кошик**: localStorage браузера.  
**База даних**: SQLite (можна перейти на PostgreSQL).
