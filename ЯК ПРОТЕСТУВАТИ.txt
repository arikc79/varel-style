1️⃣  КЛОНУВАТИ РЕПОЗИТОРІЙ:

   Відкрийте PowerShell або Command Prompt і виконайте:

   git clone https://github.com/arikc79/varel-style.git
   cd varel-style

────────────────────────────────────────────────────────────

2️⃣  ЗАПУСТИТИ АВТОМАТИЧНЕ НАЛАШТУВАННЯ:

   Просто двічі клацніть на файл:

   ► SETUP_AND_RUN.bat

   АБО через командний рядок:

   .\SETUP_AND_RUN.bat

────────────────────────────────────────────────────────────

3️⃣  ГОТОВО! 🎉

   Скрипт автоматично:
   ✅ Перевірить Python
   ✅ Створить віртуальне середовище
   ✅ Встановить всі залежності
   ✅ Налаштує базу даних
   ✅ Створить адміністратора (admin / admin123)
   ✅ Завантажить демо-дані (7 товарів + фото)
   ✅ Запустить сервер на http://127.0.0.1:8000
