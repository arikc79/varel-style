## 🚀 Як запустити:

### Варіант 1: Швидкий запуск (Windows)
Просто двічі клацніть на файл:
```

```

### Варіант 2: Вручну через термінал
```bash
cd backend
.\.venv\Scripts\python.exe manage.py runserver 127.0.0.1:8000
```

## ⚙️ Налаштування:

База даних зараз використовує SQLite для простоти. Щоб перейти на PostgreSQL:

1. Встановіть PostgreSQL
2. Відредагуйте файл `backend/.env`:
   ```env
   USE_SQLITE=False
   DB_NAME=ваша_база
   DB_USER=ваш_користувач
   DB_PASSWORD=ваш_пароль
   DB_HOST=localhost
   DB_PORT=5432
   ```
3. Запустіть міграції знову
